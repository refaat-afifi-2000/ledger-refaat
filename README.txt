# دفتر الأستاذ / رفعت - نسخة PWA

هذه نسخة أكثر احترافية من التطبيق لتعمل كـ Web App على الآيفون.

## الملفات
- index.html: التطبيق
- manifest.json: بيانات اسم التطبيق والأيقونة وطريقة الفتح
- sw.js: تشغيل كاش/أوفلاين بعد أول فتح
- icons/: أيقونات التطبيق

## طريقة النشر المجانية المقترحة
ارفع هذه الملفات كما هي على GitHub Pages أو Netlify، ثم افتح رابط الموقع من Safari على الآيفون واختر:
Share > Add to Home Screen

مهم: لا ترفع ملف ZIP نفسه، بل ارفع محتويات المجلد.
